# Seal-C
Seal-C is a lightweight library for Discord.

[Discord](https://discord.gg/XEzUV5t3j9)

## Building
Requirements:
1. Linux (Linuxmasterrace)
2. CMake
3. GCC
Building:
```
mkdir build
cd build
cmake ..
make
```
